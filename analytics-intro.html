About
什麼是 Umami？
Umami 是一個開源的網站分析平台，定位為「注重隱私的 Google Analytics 替代品」。 它不使用 Cookie、不跨站追蹤使用者，追蹤碼小於 2KB，不影響網站載入速度。 我們使用 Umami Cloud（官方託管版），只需在 HTML 中加入一行 <script> 即可開始收集數據。

How It Works
數據怎麼來？
玩家在居酒屋打開遊戲時，Umami 會自動記錄基本流量資訊。在遊戲的關鍵節點（開始、結束、切換 BGM），我們額外埋設了自訂事件，記錄遊玩細節。

Automatic
自動追蹤的指標
以下指標由 Umami 自動收集，不需要額外寫程式。開啟遊戲頁面即開始記錄。
指標說明對我們的意義
Pageviews 每個頁面被開了幾次 首頁 / 可杯 / 菊花杯 各被開了幾次
Unique Visitors 多少台不同裝置使用 最接近「多少桌客人玩過」
Visit Duration 平均停留時間 大家花多少時間在玩
Pages 各頁面流量排行 哪個遊戲最受歡迎
Devices 手機 / 平板 / 電腦 iPhone vs iPad 比例
Browsers Safari / Chrome / 其他 要優先支援哪個瀏覽器
OS iOS / Android 客群使用什麼系統
Languages 瀏覽器語系 日本客人 vs 台灣客人比例
Countries 來源地區 使用者地理分布
Hours 按小時流量分布 哪個時段最多人玩（營業高峰）
Custom Events
自訂事件追蹤
在遊戲的關鍵節點，我們埋設了自訂事件，記錄遊玩行為的細節。 這些事件會出現在 Umami Dashboard 的 Events 區塊，可以點進去看各欄位的分布統計。
🎲 可杯（Bekuhai）
bekuhai-start
觸發時機：每次按下「入座！」開始遊戲時
players 幾人遊玩 lang 語言 (zh/ja)
bekuhai-complete
觸發時機：8 回合結束，進入酒豪排行榜畫面時
players 幾人遊玩 rounds 玩了幾回合 duration 遊戲時長（秒） lang 語言 (zh/ja)
🌸 菊花杯（Kikuhai）
kikuhai-start
觸發時機：選好杯數模式後，按下「開始！」時
cups 杯數模式 (6/9/12/15) lang 語言 (zh/ja)
kikuhai-complete
觸發時機：翻到菊花，進入一輪結束畫面時
cups 杯數模式 penalty 罰酒杯數 flipped 翻了幾杯才中 duration 遊戲時長（秒） lang 語言 (zh/ja)
🥢 箸拳（Hashiken）
hashiken-start
觸發時機：按下「開始對決！」進入遊戲時
lang 語言 (zh/ja) tutorialSkipped 是否跳過教學
hashiken-complete
觸發時機：三戰兩勝結束，進入結算畫面時
winner 獲勝方 (player/ai) playerScore 玩家得分 aiScore AI 得分 rounds 總回合數 duration 遊戲時長（秒） lang 語言 (zh/ja)
🎵 BGM 偏好
bgm-switch
觸發時機：玩家點擊 BGM 切換按鈕時
game 在哪個遊戲 (bekuhai/kikuhai/hashiken) to 切到哪首 (enkai/moonlight/summer/festival)
