## 🎲 可杯（Bekuhai）

### 遊戲規則

高知傳統的「べくはい」遊戲。六角獨樂上畫有三種面具，對應三種特殊酒杯：

| 酒杯 | 容量 | 特色 | 難度 |
|------|------|------|------|
| **天狗杯** 👺 | 140ml | 長鼻擋路，必須握住鼻子喝 | ★★★★★ |
| **火男杯** 😗 | 50ml | 杯底有洞，不堵就漏酒 | ★★★ |
| **阿龜杯** 😊 | 20ml | 圓底放不下，必須一飲而盡 | ★★ |

### 遊戲流程

1. **轉獨樂**：輪到的人按住獨樂，放開旋轉
2. **唱咒語**：旋轉時全場齊唱「べろべろの神様」
3. **揭曉**：獨樂停下，指向的人＋朝上的杯型 = 喝酒命運
4. **喝酒**：被選中者用對應的酒杯喝下清酒
5. **8 回合**後，飲最多者為「酒豪」勝出

## 設計理念

### 🏮 居酒屋環境最優先

這不是一般的手機 App — 使用場景是**昏暗、吵雜、玩家半醉**的居酒屋。每個設計決策都基於這個前提：

- **字大、按鈕大**：醉酒時手指不精準，最小觸控面積 48px
- **高對比度**：`--cream-dim: #c4b496` 在昏暗燈光下仍清晰（5.5:1 對比）
- **語音引導**：切過環境噪音（volume = 1.0），BGM 壓低（gain = 0.25）
- **每頁只有一個主要動作**：降低認知負擔

### 📱 多人共享一台手機

- **可杯**：手機平放桌上，旁觀者距離 30-50cm，所以重要資訊用大字
- **菊花杯**：手機推到不同人面前，不需要輸入名字（只說「下面一位」）
- **覆蓋層（Overlay）**：關鍵時刻佔滿全螢幕，從各角度都看得到

### 🎭 3D 立體感

- **木紋圓桌**：CSS radial-gradient 模擬真實木桌俯瞰
- **座墊座位**：紅色圓形座布團，像真實日式座敷
- **真實杯子/獨樂 PNG**：使用者提供的去背實物照片
- **CSS perspective**：傾斜角度營造立體感

## 技術架構

### 單一 HTML 檔案架構

每個遊戲是一個獨立的 HTML 檔案（inline CSS + JS），沒有 build 工具、沒有框架、沒有 npm。

```
kochi-games/
├── index.html              # 首頁（遊戲選單）
├── bekuhai.html            # 可杯（~1100 行）
├── kikuhai.html            # 菊花杯（~1050 行）
├── hashiken.html           # 箸拳（~1320 行）
├── manifest.json           # PWA manifest
├── sw.js                   # Service Worker
├── bgm-kenshinso.mp3       # 健身操 BGM（3MB）— 可杯專用
├── 月夜思鄉.mp3            # 月夜思鄉 BGM（3.4MB）
├── 夏日廟會.mp3            # 夏日廟會 BGM（3.1MB）
├── 祭典夜晚.mp3            # 祭典夜晚 BGM（3.0MB）
├── icon-192.png / 512.png  # PWA icon
├── cup-*-clear.png         # 杯子去背圖（3 種）
├── cup-tengu-splash.png    # 天狗杯 GameOver 版
├── koma-1/2/3.png          # 六角陀螺（3 角度）
```

```
AudioManager
├── BGM（全部 MP3，統一 volume=0.08）
│   ├── enkai — 🏋️ 健身操（可杯專用）
│   ├── moonlight — 🌙 月夜思鄉
│   ├── summer — 🏮 夏日廟會
│   └── festival — 🎆 祭典夜晚
├── SFX（Web Audio API 合成）
│   ├── playClick() — UI 點擊
│   ├── playSpinStart() / playSpinStop() — 獨樂旋轉
│   ├── playFlipCup() — 翻杯陶瓷聲
│   ├── playDrinkGulp() — 喝酒咕嚕聲
│   └── playWin() / playSake() — 慶祝
└── 自動 Duck（語音播放時 BGM 降至 0.01）

VoiceManager
├── jaVoice — Kyoko (ja-JP)
├── zhVoice — Meijia (zh-TW)
├── speak() — 播放語音
├── speakAsync() — 播放語音（等待完成）
└── unlockSpeech() — 首次互動時解鎖
```
